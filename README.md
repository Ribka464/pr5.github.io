# Student Task
**Student Task** - это приложение, разработанное для помощи студентам в организации их учебного процесса. Система позволяет управлять задачами, отслеживать сроки их выполнения и анализировать свою продуктивность.

---

## Основные функции (модули)

### 1. Модуль аунтификации
- Регистрация нового пользователя
- Авторизация (вход и выход из системы)

### 2. Модуль управления задачами
- **Создание задачи** - возможность добавить новую задачу.
- **Атрибуты задачи** - у каждой задачи есть нахвание описание, предмет и срок сдачи.
- **Изменение статуса** - задачи можно помечать как "В процессе" или "Выполнено".
- **Просмотр и фильтрация** - возможность просматривать все задачи и фильтровать их по предмету или статусу.

### 3. Модуль отчётности(статистики)
- **Статистика** - просмотр простой статистики.
